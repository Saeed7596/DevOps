**Note**: 
* `request` : The minimum resource that a Pod requires to run.
* `limit` : The maximum resource that the Pod is allowed to consume. 
* Always Request < Limit
```yaml
kind: ResourceQuota
apiVersion: v1
metadata:
  name: quota-name
  namespace: namespace-name
spec:
  hard:
    limits.cpu: '32'
    limits.memory: 68Gi
    pods: '15'
    requests.cpu: '12'
    requests.memory: 16Gi
```
